# 스마트팜 시뮬레이터 배포 가이드

## 다른 PC에서 실행하기 위해 필요한 파일들

### 1. C# 애플리케이션 파일 (bin/Debug 또는 bin/Release 폴더 전체)

```
bin/Debug/ (또는 bin/Release/)
├── farm.exe                    # 메인 실행 파일
├── farm.exe.config             # 설정 파일
├── farm.pdb                    # 디버그 심볼 (선택사항)
├── Newtonsoft.Json.dll         # JSON 라이브러리
├── Newtonsoft.Json.xml         # XML 문서
├── TwinCAT.Ads.dll             # TwinCAT PLC 통신 라이브러리
├── TwinCAT.Ads.xml             # XML 문서
├── sensor_thresholds.txt       # 센서 임계값 설정 (자동 생성 가능)
└── farm_info.txt               # 팜 정보 설정 (자동 생성 가능)
```

**중요**: `bin/Debug` 또는 `bin/Release` 폴더 전체를 복사하세요.

### 2. Flask 서버 파일 (C:\farmui 폴더 전체)

```
C:\farmui\
├── app.py                      # Flask 메인 서버
├── ai_analysis.py              # AI 분석 모듈
├── crop_config.py              # 작물 설정 파일
├── data_manager.py             # 데이터 관리 모듈
├── ml_trainer.py               # 머신러닝 훈련 모듈
├── requirements.txt            # Python 패키지 목록
├── templates/
│   └── index.html              # 웹 인터페이스
└── models/                     # AI 모델 저장 폴더 (선택사항)
```

**중요**: `C:\farmui` 폴더 전체를 복사하세요. (경로는 변경 가능)

---

### 3. 필요한 소프트웨어 설치

#### Windows PC에 설치해야 할 것들:

1. **.NET Framework 4.7.2 이상**
   - 다운로드: https://dotnet.microsoft.com/download/dotnet-framework
   - 또는 Windows Update를 통해 설치

2. **Python 3.8 이상**
   - 다운로드: https://www.python.org/downloads/
   - 설치 시 "Add Python to PATH" 옵션 체크

3. **TwinCAT ADS 라이브러리** (PLC 통신용)
   - TwinCAT 설치 시 자동 포함
   - 또는 TwinCAT.Ads.dll을 bin/Debug 폴더에 복사하면 됨 (이미 포함됨)

---

### 4. 설치 및 실행 방법

#### Step 1: C# 애플리케이션 배포

1. `bin/Debug` (또는 `bin/Release`) 폴더 전체를 새 PC로 복사
2. 원하는 위치에 폴더를 배치 (예: `D:\SmartFarm\`)
3. `farm.exe` 실행

#### Step 2: Flask 서버 설정

1. `C:\farmui` 폴더를 새 PC로 복사 (경로는 변경 가능)
2. 명령 프롬프트 또는 PowerShell을 관리자 권한으로 실행
3. Flask 서버 폴더로 이동:
   ```cmd
   cd C:\farmui
   ```

4. Python 패키지 설치:
   ```cmd
   pip install -r requirements.txt
   ```

5. Flask 서버 실행:
   ```cmd
   python app.py
   ```

   또는 백그라운드 실행:
   ```cmd
   pythonw app.py
   ```

#### Step 3: 웹 인터페이스 접속

- 브라우저에서 `http://localhost:5000` 접속

### 6. 주의사항

1. **경로 변경 시**
   - Flask 서버 경로를 변경한 경우, C# 애플리케이션의 Flask 서버 URL도 변경해야 함
   - `Form1.cs` 파일의 65번째 줄에서 Flask 서버 URL 수정:
     ```csharp
     private const string FlaskServerUrl = "http://localhost:5000";
     ```
   - 다른 PC의 IP 주소를 사용하려면 `localhost` 대신 IP 주소 사용 (예: `http://192.168.1.100:5000`)

2. **포트 충돌**
   - Flask 서버는 기본적으로 포트 5000 사용
   - 다른 프로그램이 5000 포트를 사용 중이면 `app.py`에서 포트 변경 필요

3. **PLC 연결**
   - TwinCAT PLC가 연결되어 있어야 정상 작동
   - PLC가 없는 경우 시뮬레이션 모드로 동작 가능 (코드 확인 필요)

4. **방화벽 설정**
   - Flask 서버가 외부에서 접근 가능하도록 하려면 방화벽 설정 필요
   - 로컬에서만 사용하는 경우 불필요

### 8. 문제 해결

#### Flask 서버가 실행되지 않는 경우
- Python이 설치되어 있는지 확인: `python --version`
- 패키지가 설치되어 있는지 확인: `pip list`
- 포트가 사용 중인지 확인: `netstat -ano | findstr :5000`

#### C# 애플리케이션이 실행되지 않는 경우
- .NET Framework 버전 확인
- TwinCAT.Ads.dll이 같은 폴더에 있는지 확인
- 관리자 권한으로 실행 시도

#### 웹 인터페이스가 표시되지 않는 경우
- Flask 서버가 실행 중인지 확인
- 브라우저 콘솔에서 오류 확인 (F12)
- CORS 설정 확인
